# AriaNg-go
AriaNg in go http server


# Help

```
$ ./AriaNg-go -h
Usage of ./AriaNg-go:
  -a string
    	bind address. (default ":18080")
```

visit `http://localhost:18080`

change the defalt bind address:
```
$ ./AriaNg-go -a=":80"
```
